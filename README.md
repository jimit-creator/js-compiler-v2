# JavaScript Online Compiler

A responsive JavaScript online compiler web app built with React, TypeScript, and Tailwind CSS. This application allows users to write, execute, and share JavaScript code snippets directly in the browser.

![JavaScript Online Compiler](https://example.com/screenshot.jpg)

## Features

- 🖥️ Responsive design for desktop, tablet, and mobile devices
- 📝 Code editor with syntax highlighting using Monaco Editor
- ▶️ Run button to execute JavaScript code
- 📋 Console output display for results and errors
- 🌗 Dark/light mode toggle
- ↔️ Resizable editor panels
- 🔗 Quick Share functionality for social media
- 📱 Mobile-friendly interface

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Monaco Editor
- React Context API
- TanStack Query
- Lucide icons

### Backend
- Express
- Node.js VM for code execution
- In-memory storage
